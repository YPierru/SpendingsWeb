Data Flow
---------
1. User clicks "Load CSV Data"
2. ExpenseContext.loadFromCSV() triggered
3. csvParser.parseCSVFile() fetches and parses CSV
4. Validators validate each row
5. dataProcessing.calculateSummary() generates statistics
6. localStorage.saveExpenses() persists data
7. UI updates with summary data
8. On page reload: localStorage.loadExpenses() restores data
